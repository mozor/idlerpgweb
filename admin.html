IdleRPG Administrator Commands

This is not the full list of commands for the Idle RPG bot, but only the list of admin commands.

INFO

Usage: /msg IdleBot INFO

retrieve some fairly useless stats about the bot.

DIE

Usage: /msg IdleBot DIE

kills the bot

HOG

Usage: /msg IdleBot HOG

Summons the Hand of God. Performs a calamity or helps a single player

RESTART

Usage: /msg IdleBot RESTART

Restarts the bot

CHPASS

Usage: /msg IdleBot CHPASS [user] [password]

Changes a characters password in the database

CHCLASS

Usage: /msg IdleBot CHCLASS [user] [class]

Changes a character's class in the database.

CHUSER

Usage: /msg IdleBot CHUSER [user] [new user]

Changes a user's name in the database. Should be avoided. Just have the user register a new account and delete the old one.

PUSH

Usage: /msg IdleBot PUSH [user] [seconds]

Push a player toward his/her goal by subtracting time from the next time to level. A negative time can be inserted to penalize a user. Please note, you can cause a user to get negative time which will cause issues.

DEL

Usage: /msg IdleBot DEL [user]

Deletes the user from the database

FSAY

Usage: /msg IdleBot FSAY [message]

Says the message in the channel, to avoid penalties

JUMP

Usage: /msg IdleBot JUMP [server][:port]

Connects the bot to the specified server

SILENT

Usage: /msg IdleBot SILENT [mode]

Switch the bot between these modes:
0 - bots send all privmsgs
1 - Disables messages to the channel
2 - Disables private messages and notices to users
3 - Disables private messages and notices to users and channels

BACKUP

Usage: /msg IdleBot BACKUP

tell bot to copy $opts{'dbfile'} to .dbbackup/$opts{'dbfile'}

PAUSE

Usage: /msg IdleBot PAUSE

Enables/disables pause mode

PEVAL

Usage: /msg IdleBot PEVAL [code]

execute arbitrary argument as Perl code. Queues output > 3 lines or >1k of text. Some useful PEVAL commands:

  • Delete all accounts not logged in in 4 weeks (See also: DELOLD):
    /msg bot PEVAL delete $rps{$_} for grep { time()-$rps{$_}{lastlogin} > 3600*24*7*4 && !$rps{$_}{online} } keys %rps;
  • Remove one hour from everyone's clocks:
    /msg bot PEVAL $rps{$_}{next} -= 3600 for keys %rps;
  • List all online users, separated by commas:
    /msg bot PEVAL join(', ',grep { $rps{$_}{online} } keys %rps);
  • View contents of a file on remote host:
    /msg bot peval `cat file`
  • Turn on debug mode:
    /msg bot peval $opts{debug}=1;
  • Force write-out of database:
    /msg bot peval writedb();

DELOLD

Usage: /msg IdleBot DELOLD [days]

Removed accounts for users not logged in in the specified number of days.

CLEARQ

Usage: /msg IdleBot CLEARQ

Clears the outgoing message queue.

MKADMIN

Usage: /msg IdleBot MKADMIN [user]

Sets the specified user as an administrator
